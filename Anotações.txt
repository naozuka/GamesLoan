- Para criar uma nova solução:
dotnet new sln -n <SolutionName>

- Para listar as sdk's existentes:
dotnet --list-sdks

- Para adicionar o global.json (informando a sdk do projeto):
dotnet new globaljson --sdk-version 5.0.103

- Adicione os projetos no global.json

{
    "projects": ["src", "test"],
    "sdk": {
        "version": "5.0.103"
    }
}

- Para criar um projeto de testes:
dotnet new xunit -n <ProjectName>.Tests

- Para adicionar os projetos na solução:
dotnet sln add <ProjectPath>/<ProjectName>.csproj

- Instale a extensão de testes:
.net Core Test Explorer (Juan Han - Text Explorer for .NET Core(MSText, xUnit, NUnit)

- Na pasta da solução, crie um arquivo settings.json e adicione o seguinte:
Obs: primeira linha aponta para o projeto de testes e a segunda para não enviar dados.
{
    "dotnet-test-explorer.testProjectPath": "<ProjectPath>/<ProjectName>.csproj,
    "dotnet-test-explorer.enableTelemetry": false
}
